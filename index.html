FAQ

Questions

Is my screen content private?

Yes. Screenshots are sent directly to Anthropic's API over TLS and are never stored or logged by Opsis. Anthropic's API usage policies apply.

How does the credit system work?

Each screen analysis uses 1 credit. Free accounts start with 10 credits. Pro plans get 1,000 credits per month that reset each billing cycle.

What Windows versions are supported?

Windows 10 (version 1903+) and Windows 11. The app requires the WebView2 runtime — it's pre-installed on Windows 11 and available as a free download for Windows 10.

Do unused credits expire?

Credit pack credits never expire. Monthly plan credits reset at the start of each billing cycle and do not roll over.

What AI model does Opsis use?

The Default model uses Claude Haiku for fast, lightweight responses. The Pro model (available on paid plans) uses Claude Sonnet for deeper, more accurate analysis.
